Características principales

Todo lo que necesitas para gestionar el control de asistencia de tu equipo

Geolocalización inteligente

Cada fichada registra la ubicación GPS exacta del empleado. Perfecto para equipos en la calle, vendedores y personal de campo.

Gestión de novedades

Vacaciones, enfermedades, estudios, permisos. Totalmente configurable según tus necesidades. Define tus propios tipos de novedades.

Aprobaciones configurables

Define flujos de aprobación personalizados. Supervisores pueden aprobar o rechazar fichadas y novedades.

Dashboard en tiempo real

Visualiza quién está fichado ahora, reportes de asistencia, horas trabajadas y análisis de ausentismo en tiempo real.

Legajos completos

Gestiona datos personales, familiares, laborales, documentos y más. Todo centralizado en un solo lugar con campos personalizables.

Biometría avanzada

Reconocimiento facial integrado en la app o biométrico con accesorio para fichadas ultrarrápidas y seguras.

Todo tipo de fichadas

Entrada, salida, breaks, horas extras y cualquier tipo personalizado que necesites. Sistema 100% flexible.

Integración total

API REST completa para integrar con cualquier sistema. Conecta con Tango, SAP, Odoo o tu ERP personalizado.

Tablas maestras flexibles

Define lugares, puestos de trabajo, obras sociales y cualquier categoría que necesites. Sistema totalmente parametrizable.

Evaluación de desempeño

Crea formularios de evaluación personalizados, define KPIs, programa evaluaciones periódicas y genera reportes de desempeño del personal.

Onboarding digital

Incorpora nuevos empleados con flujos automatizados: documentación, capacitaciones, asignación de recursos y seguimiento del proceso completo.
